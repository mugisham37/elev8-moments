# Elev8 Rwanda Website

This is the official website for Elev8 Rwanda, a brand with two sub-brands: Elev8 Media and Elev8 Moments.

## Features

- Modern, responsive design with dark mode support
- Comprehensive pages for Home, About, Brands, Blog, and Contact
- Dedicated pages for each sub-brand (Elev8 Media and Elev8 Moments)
- Built with Next.js and Tailwind CSS for optimal performance
